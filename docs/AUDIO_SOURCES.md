## QUY ƯỚC ĐẶT TÊN (cho bổ sung sau)

- Snake_case tiếng Việt không dấu
- Subfolder theo loại: `sung/`, `can_chien/`, `buoc_chan/`, `luu_dan/`, `bom/`, `music/`
- Trong `weapon_data.gd`, field `am_thanh_ban`:
  - File trong `sung/`: chỉ tên file (vd `"ak47.wav"`)
  - File ngoài `sung/`: tên đầy đủ với folder (vd `"can_chien/dao_dam.wav"`)

## KHI THÊM ÂM THANH MỚI

1. Tải file → đặt tên đúng quy ước → bỏ vào subfolder phù hợp
2. **Cập nhật file này** với hàng mới (file, license, tác giả, nguồn)
3. Nếu là CC-BY hoặc CC-BY-SA, thêm vào credits screen
4. Cập nhật `weapon_data.gd` field `am_thanh_ban` cho khẩu liên quan
5. Commit với message `feat(audio): them tieng [...]`
